# API Reference

This section provides the full API reference for the ZarrNii library, organized by module.

| Module | Description |
|--------|-------------|
| [zarrnii](zarrnii.md) | Core `ZarrNii` class, top-level utilities, and exceptions |
| [zarrnii.analysis](analysis.md) | Image analysis functions (histograms, density estimation, region properties) |
| [zarrnii.transform](transform.md) | Affine and displacement field transforms (`AffineTransform`, `DisplacementTransform`) |
| [zarrnii.atlas](atlas.md) | Atlas handling (`ZarrNiiAtlas`) |
